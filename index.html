Projeto pronto
para grandes telas

A página inicial do site foi montada com classes Large e Extra Large do Bootstrap para que o usuário tenha uma ótima experiencia em telas maiores.

Responsividade, Fontes e Ícones

Assim como para telas grandes, usamos classes Extra small, Small Medium do Bootstrap para que o site e sistema possam ser facilmente utilizados em tablets e celulares de tamanhos variados. Fontes e icones utilizados do Bootstrap e do Google para uma maior compatibilidade.
